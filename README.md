Modern sales is broken. Teams spend 70% of their time on manual research, lead scoring, and drafting emails that get ignored. **SalesOS** transforms your sales process into a high-signal, autonomous intelligence terminal.

By combining **Exa-powered real-time research** with **multi-agent orchestration**, SalesOS doesn't just automate tasks—it executes high-fidelity sales operations with institutional precision.

## Core Intelligence Modules

### 🎯 1. Autonomous Lead Qualification
Stop chasing dead ends. Our **Lead Qualifier Crew** uses Exa to perform deep-web research on every inbound lead, scoring them against your custom ICP criteria in seconds.
- **Benefit**: Focus 100% of your energy on high-intent deals.
- **Proof**: Scans and scores 50+ leads in under 2 minutes.

### ✉️ 2. Hyper-Personalized Personalization
Generic templates are for the 2010s. The **Cold Personalizer** researches recent news, LinkedIn profiles, and company updates to craft "Institutional but helpful" outreach.
- **Benefit**: 3x your reply rates with hooks that actually demonstrate you've done the homework.

### 📝 3. RAG-Powered Proposal Drafting
Drafting complex proposals used to take hours. Our **Proposal Drafter** uses RAG (Retrieval-Augmented Generation) to pull the perfect project templates and technical specs for every unique deal.
- **Benefit**: Send high-fidelity proposals before the follow-up meeting even ends.

## Institutional Safety & Guardrails
We believe in **Human-in-the-Loop (HITL)** sovereignty. SalesOS is built with robust safety features to ensure your brand is never at risk.

- **Dry-Run Mode**: Toggle a global safety switch to simulate all outbound actions without sending a single byte.
- **Actions Pending Approval**: A centralized dashboard carousel allows you to review and edit every AI-generated output before it goes live.
- **Cost Guardrails**: Integrated token budgeting at the Edge Function level to prevent runaway API costs.

## Quick Start (3-Minute Setup)

### 1. Deploy the Backend
Click below to deploy your private SalesOS instance to Supabase:
[Deploy to Supabase](https://supabase.com/dashboard/new)

### 2. Configure Environment
Clone this repo and add your keys to `.env.local`:
```bash
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
EXA_API_KEY=your_exa_key
```

### 3. Launch the Terminal
```bash
npm install
npm run dev
```
Navigate to `http://localhost:3000` to enter the command center.
